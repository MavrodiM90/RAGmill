Тесты регулярок

Как пользоваться

Auto-detect источника

Скрипт сам определяет тип документа по характерным признакам: UUID-паттерны → Notion, frontmatter + [[wiki]] → Obsidian, тег <html> → HTML, поле cells → Jupyter. Кликните на бейдж «Источник» рядом с заголовком, чтобы переопределить вручную.

Три режима

Очистить — убирает мусор из исходника, сохраняя структуру MD.

Разбить на чанки — нарезает документ на куски с метаданными для векторной БД. Стратегии: по заголовкам (с breadcrumbs), по абзацам, скользящее окно с перекрытием, рекурсивный сплиттер à la LangChain.

Plain text — снимает всю разметку, оставляя чистый текст.

Просмотр результата

Три view-режима справа: Сырой (исходный код), Превью (отрендеренный MD или карточки чанков), Diff (что именно изменилось).

Хоткеи

Ctrl+Enter — обработать заново · Ctrl+S — скачать результат · Ctrl+Shift+C — скопировать · все настройки сохраняются автоматически

Файлы

Перетащите .md, .markdown, .txt, .html, .ipynb прямо на текстовое поле. При нескольких файлах они склеиваются с разделителями. Ничего не уходит на сервер — обработка в браузере.

Выбор источника

Auto-detect определяет тип сам, но можно переопределить вручную. Каждый источник применяет специфичные правила нормализации.
